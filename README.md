# Генератор QR-Кодов

## О проекте

Данный проект реализует генератор кодов на языке программирования Python. Из зависимостей используются colorama (для цветного вывода текста), qrcode (для генерации qr-кодов) и pillow. В проекте присутствует разделение на логические модули:

* main.py — главный файл для запуска приложения
* functions.py — основной код программы, содержит разные функции для выполнения основного функционала
* tests.py — проверка написанного кода с помощью системы текстов (assert)
* requirements.txt — зависимости, необходимые для развёртывания проекта
* requirements-dev.txt — зависимости, нужны исключительно для разработки проекта
* .gitignore и .pylintrc — служебные файлы

К функциям проекта можно отнести:

* Генерация QR-Кода по тексту
* Генерация QR-Кода по E-Mail
* Генерация QR-Кода по сайту
* Генерация QR-Кода по MECARD (карточке сотрудника)
* Возможность регулировать уровень коррекции ошибок (L/M/Q/H)

## Инструкция по развёртыванию

### Для использования проекта

1. Установите нужны зависимости: `pip install -r requirements.txt`
2. Запустите проект: `python main.py`

### Для разработки (для разработчиков)

1. Установите нужны зависимости: `pip install -r requirements.txt requirements-dev.txt`
2. Разрабатывайте проект!
3. Запустите проект: `python main.py`

### Требования к запуску

* Python 3.10+
* Установленные зависимости (colorama, qrcode и pillow)
